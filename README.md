## Taskify
# Project Description

Taskify is a full-stack todo application designed to help users organize and manage their daily tasks efficiently. The project features a modern React + Vite frontend styled with TailwindCSS, and a planned Express.js + MongoDB backend for secure data storage and user management.

Currently, the app supports:

Adding, editing, completing, and deleting tasks

Interactive and responsive UI

Local state management for todos

Planned features include:

Secure user authentication (Login & Signup with JWT)

Persistent storage with MongoDB Atlas

User-specific tasks with full CRUD operations via REST API

Categories, priorities, search, and drag-and-drop reordering

⚠️ Note: The backend (Express + MongoDB) and authentication system (Login/Signup) are still under development. Right now, todos are managed on the frontend only.

# 🚀 Tech Stack

Frontend: React, Vite, TailwindCSS, Axios

Backend (Planned): Node.js, Express.js, MongoDB, JWT

Deployment (Planned): Netlify/Vercel (frontend), Render/Heroku (backend)
